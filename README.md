# ⚗️ ХимСайт v2

Интерактивный сайт по общей химии. React + Vite + Tailwind CSS.

## Запуск

```bash
npm install
npm run dev
```

## Разделы

- 📚 **Теория** — 8 разделов общей химии
- 🧬 **Таблица Менделеева** — все 118 элементов с карточками
- ⚗️ **Лаборатория** — виртуальные химические реакции
- 🧠 **Квиз** — 3 уровня сложности
